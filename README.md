## Quick Start

### 1. Install dependencies
```bash
npm install
```

### 2. Set up Supabase

You'll need to create a Supabase project and run the SQL schema. See `SUPABASE_SETUP.md` for details.

### 3. Run locally
```bash
npm run dev
```

Visit `http://localhost:3000`

### 4. Build for production
```bash
npm run build
```
